# Term Deposit Calculator

## Prerequisites

- **Node.js** version **22.14.0**

## install dependencies

- npm install

## to run locally

- npm run dev

## to run test

- npm run test

## Input constraints

- deposit Amount: 1000 - 1500000
- interest Rate (%): 0 - 15
- investment Term (month): 12 - 120
- interest Payment Frequency: 1. Monthly, 2. Quarterly, 3. Annually , 4. At Maturity
